# Installation

You can install XState using your favorite package manager or embed the `<script>` directly from a CDN.

<Tabs>
<TabItem value="yarn" label="yarn">

```bash
yarn add xstate
```

</TabItem>

<TabItem value="npm" label="npm">

```bash
npm install xstate
```

</TabItem>
</Tabs>

## CDN

You can import XState from the [Skypack CDN](https://www.skypack.dev/xstate@beta):

```ts
import { createMachine, interpret } from 'https://cdn.skypack.dev/xstate@beta';

const machine = createMachine({
  initial: 'a',
  states: {
    a: {},
  },
});

const actor = interpret(machine);
actor.subscribe((state) => {
  console.log(state);
});
actor.start();
```
