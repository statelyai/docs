# Cypress

<!-- deps: ["testing-intro"] -->

Integrating with [Cypress](https://www.cypress.io) is simple with `@xstate/test`.

```js
import { createTestMachine, createTestModel } from '@xstate/test';

const machine = createTestMachine({
  // machine config
});

describe('My app', () => {
  createTestModel(machine)
    .getPaths()
    .forEach((path) => {
      it(path.description, () => {
        path.testSync({
          states: {},
          events: {},
        });
      });
    });
});
```
